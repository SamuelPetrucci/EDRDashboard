# Dashboard Theme Update - Orange & Light Navy Blue

## New Color Scheme

### Primary Colors
- **Orange**: `#FF6B35` (Vibrant Orange) - Primary accent color
- **Orange Dark**: `#E85A2B` - For gradients and hover states
- **Orange Light**: `#FF8C5A` - For lighter accents

### Navy Colors
- **Light Navy Blue**: `#4A90E2` - Primary navigation and links
- **Navy Dark**: `#357ABD` - Headers and emphasis
- **Navy Light**: `#6BA3E8` - Hover states and highlights

### Status Colors (Maintained)
- **Resilient (Green)**: `#10B981` - 80-100% scores
- **Restoring (Amber)**: `#F59E0B` - 60-79% scores
- **Need Support (Red)**: `#EF4444` - 0-59% scores

### Neutral Colors
- **Background**: `#F8FAFC` - Very light gray-blue
- **Card Background**: `#FFFFFF` - Pure white
- **Text Primary**: `#1E293B` - Dark slate
- **Text Secondary**: `#64748B` - Medium slate
- **Borders**: `#E2E8F0` - Light slate

## Design Improvements

### Consistency Updates
1. **Shadows**: Standardized shadow system (sm, md, lg)
2. **Border Radius**: Consistent 12px for cards, 10px for items
3. **Transitions**: Smooth 0.3s ease transitions throughout
4. **Hover Effects**: Consistent transform and shadow effects
5. **Gradients**: Orange and Navy gradients used consistently

### Visual Enhancements
- **Cards**: Subtle borders, consistent shadows, smooth hover effects
- **Buttons**: Gradient backgrounds with hover animations
- **Icons**: Gradient backgrounds matching theme
- **Badges**: Rounded with shadows for depth
- **Links**: Navy blue with orange hover states

### Component-Specific Updates

#### Header & Navigation
- Navy blue gradient background
- Orange accent for active navigation items
- Smooth hover transitions

#### Cards & Sections
- Consistent border radius (12px)
- Subtle borders and shadows
- Smooth hover lift effects
- Gradient backgrounds on stat cards

#### Buttons & Links
- Orange gradient for primary actions
- Navy blue for secondary links
- Smooth hover animations
- Consistent padding and sizing

#### Status Indicators
- Color-coded badges with shadows
- Consistent sizing and spacing
- Clear visual hierarchy

## Color Usage Guidelines

### Orange (`#FF6B35`)
- Primary action buttons
- Active navigation states
- Important highlights
- Stat card icons (alternating)
- Scorecard accents

### Light Navy Blue (`#4A90E2`)
- Navigation background
- Primary links
- Headings
- Parish names
- Domain titles

### Gradients
- **Header**: Navy gradient (navy-color → navy-dark)
- **Footer**: Navy gradient (navy-dark → navy-color)
- **Stat Cards**: Alternating orange and navy gradients
- **Buttons**: Orange gradient (primary → primary-dark)

## Typography
- **Headings**: Navy dark (`#357ABD`) with 700 weight
- **Body Text**: Dark slate (`#1E293B`)
- **Secondary Text**: Medium slate (`#64748B`)
- **Links**: Navy blue with orange hover

## Spacing & Layout
- Consistent padding: 1.75rem for cards, 2rem for sections
- Standardized gaps: 1.5rem between elements
- Uniform border radius: 12px cards, 10px items, 20px badges

## Interactive Elements
- **Hover States**: 
  - Cards: Lift effect (translateY -4px to -6px)
  - Buttons: Scale and shadow increase
  - Links: Color change to orange
- **Transitions**: All 0.3s ease for smooth animations
- **Shadows**: Progressive shadow system for depth

## Accessibility
- High contrast maintained for readability
- Color not used as sole indicator (text labels included)
- Consistent focus states
- Clear visual hierarchy
